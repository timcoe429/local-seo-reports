## How to Create a New Client Report (5 Minutes)

### Step 1: Copy the Template
```
1. Copy: client_template.html â†’ [client_name]_report.html
   Example: client_template.html â†’ american_dumpster_report.html
```

### Step 2: Edit the File
Open the new file in **any text editor** (Notepad, VS Code, Word, etc.)

### Step 3: Find & Replace
Use **Ctrl+H** (Find & Replace) to replace all the placeholders:

| Find This | Replace With | Example |
|-----------|--------------|---------|
| `[COMPANY_NAME]` | Client's business name | `American Dumpster Co.` |
| `[REPORT_DATE]` | Report date | `October 2025` |
| `[CURRENT_RANKING]` | Their ranking position | `#3` |
| `[GOOGLE_REVIEWS]` | Number of Google reviews | `178` |
| `[AVG_RATING]` | Average star rating | `5.0` |
| `[DOMAIN_AUTHORITY]` | Domain Authority score | `11/100` |
| `[TOTAL_BACKLINKS]` | Total backlinks | `856` |
| `[MONTHLY_TRAFFIC]` | Monthly organic traffic | `354` |

### Step 4: Add Screenshots (Optional)
Replace screenshot URLs:
```
Find: [DOMAIN_AUTHORITY_SCREENSHOT]
Replace: https://your-image-url.com/screenshot.jpg
```

### Step 5: View Report
```
1. Save the file
2. Double-click the HTML file
3. Opens in your browser
4. Press Ctrl+P to print to PDF
```
